# GrocerEase

GrocerEase is a full-stack grocery shopping platform designed to provide a seamless online shopping experience. Users can browse grocery products, manage their shopping cart, authenticate securely, and place orders through a modern and responsive interface.


## Overview

GrocerEase simplifies grocery shopping by allowing users to purchase daily essentials online. The platform offers a clean interface, secure authentication, category-based browsing, cart management, and an efficient checkout process.



## Features

- User Registration and Login
- Browse Grocery Categories
- Fruits, Vegetables, Dairy Products, Snacks, Beverages, and Sweets
- Add Products to Cart
- Update Cart Quantity
- Secure Checkout
- Delivery Address Management
- Dark and Light Theme
- Multi-language Support (English, Telugu, Hindi)
- Responsive User Interface

## Installation

### Clone the Repository

```bash
git clone https://github.com/eekshitha06/GrocerEase.git
```

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd backend
npm install
npm run dev
```
